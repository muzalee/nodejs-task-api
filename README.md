
## NodeJS API for Task Management with MongoDB

Clone the project

```bash
  git clone https://github.com/muzalee/nodejs-task-api.git
```

Go to the project directory

```bash
  cd nodejs-task-api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
  or 
  node .
```

Setup linting

```bash
  npm init @eslint/config
```

```bash
  npm run lint
  or
  npm run lint-fix
```
